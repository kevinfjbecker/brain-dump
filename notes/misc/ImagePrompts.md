# Image Creator Prompts

Below are the propts that were used to create the exmple images on the [Microsoft Designer Image Creator](https://designer.microsoft.com/image-creator) page.

I feel like there's *art* in that level of descriptive precision. In any case they are *weird* to read.

## Code

The text was scraped from the browser console with the following JavaScript

``` JavaScript
[...document.querySelectorAll('[class^="textWrapper-"]')].map(e => e.innerText)
```

and cleaned with the this mapping chain

``` JavaScript
.map(s => s.replace('\nEdit prompt', ''))
.map(s => s.slice(1, s.length-1)) // remove quotation marks
.map(s => s.trim())
```
